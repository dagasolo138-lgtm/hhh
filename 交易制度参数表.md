# 上交所黄金ETF模拟器 · 交易制度参数表

> 版本：第四版
>
> 本表定义模拟器当前的制度边界。机器可读参数以 `市场/交易制度参数.js` 为唯一来源；运行时架构以 `codex.md` 为准。

---

## 1. 模拟对象

项目是**上交所黄金ETF通用制度框架下的中国黄金现货ETF模拟器**，不绑定现实基金代码。

```text
上海黄金交易所黄金现货实盘合约
→ 境内黄金现货价格
→ 黄金ETF理论净值
→ 上交所二级市场交易价格
```

二级市场交易、现金申赎、黄金现货合约申赎为独立通道。

## 3. 开盘集合竞价

### 3.1 生命周期

```text
09:15–09:20
玩家限价单可提交、可撤销
五类自动资金根据目标仓位生成开盘限价单

09:20–09:25
玩家可继续提交，撤单锁定
自动订单按最新目标仓位刷新后锁定

09:30
玩家订单 + 自动订单 + 做市背景深度
→ 选择单一开盘价
→ 统一清算
→ 玩家剩余订单转入连续竞价中心簿
→ 自动剩余订单登记为连续竞价接续意图
```

集合竞价不接受市价单。

### 3.2 开盘价选择

候选价选择顺序：

1. 最大可成交量；
2. 最小未成交量；
3. 与前收盘价偏离最小；
4. 最小报价单位内的唯一价格。

所有开盘成交统一使用该价格。集合竞价包含自成交保护。

### 3.3 清算与接续

- 玩家成交更新现金、持仓、成本、已实现盈亏与冻结资产。
- 自动资金和做市商成交更新现金、持仓、成本与已实现盈亏。
- 玩家未成交部分保留原订单与冻结资产，转入连续竞价。
- 自动未成交部分在下一刻按连续竞价规则重新报价；当前不保留原开盘限价。
- 玩家开盘成交与转连续均写入可见日志。

## 4. 连续竞价中心订单簿

参与者：做市商、价值套利、趋势资金、散户群体、风险控制资金、先行信号资金、玩家市价单和玩家限价单。

```text
买单：价格高者优先；同价按中心委托序号早者优先。
卖单：价格低者优先；同价按中心委托序号早者优先。
```

- 玩家市价单以当日涨跌停边界形成即时有效订单，未成交部分当刻失效。
- 玩家限价单冻结资金或份额后进入中心簿，当日有效、跨日失效。
- 市价单失效残余会从中心盘口快照删除。
- 自动、做市与玩家成交统一清算双方现金、持仓、平均成本与已实现盈亏。
- 同一参与者交叉订单触发自成交保护，后提交订单被拒绝。

订单状态：

```text
待撮合 → 部分成交 → 全部成交
待撮合/部分成交 → 已撤销
待撮合/部分成交 → 已过期
```

## 5. 涨跌幅与冻结资产

```text
涨停价 = 前收盘价 × 1.10
跌停价 = 前收盘价 × 0.90
```

- 委托价格必须在当日边界内且为 0.001 元整数倍。
- 限价买单冻结：`委托价 × 剩余数量`。
- 限价卖单冻结：`剩余数量`。
- 成交、撤单、过期、拒绝都同步调整冻结资产。
- 一致性检查会核对冻结资产与待处理订单、连续订单与中心簿映射、基础盘口边界与守恒。

---

## 6. 申购与赎回

现金申赎、黄金现货合约申赎、每日清单、额度、清算与交收尚未向普通模拟玩家开放。它们不能与普通二级市场买卖混为一条路径。

---

## 7. 合规边界

- 不连接真实券商、交易所或基金申赎系统。
- 不发送真实订单。
- 不构成投资建议、收益承诺或交易指令。
- 不向玩家提供现实非公开信息。
- “先行信号”仅模拟信息扩散与流动性风险。
