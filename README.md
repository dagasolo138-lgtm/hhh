# 中国黄金模拟市场

> **每次开发完成后，必须在 `codex.md` 末尾追加版本更新概述。**

这是一个部署在 GitHub Pages 的纯前端、单标的中国黄金模拟市场。

唯一可交易资产是人民币计价的“中国黄金现货指数基金”模拟份额。交易价格来自国际现货黄金、美元兑人民币、国内黄金溢价、基金理论净值、交易者订单、集合竞价和连续竞价机制。

完整架构、变量定义、因果图、文件职责、存档规则、测试规则、已知限制和版本历史，以 [`codex.md`](./codex.md) 为准。交易制度以 [`交易制度参数表.md`](./交易制度参数表.md) 和 `市场/交易制度参数.js` 为准。

## 当前能力

- 国际金价 → 汇率 → 人民币换算金价 → 国内现货 → 基金理论净值链路。
- 理论净值与二级市场交易价格分离。
- 价值套利、趋势资金、散户群体、风险控制资金、先行信号资金与做市商。
- 十档做市盘口、价差、深度、撤单率、补单率、订单失衡和价格冲击。
- 当日回转、0.001元报价单位、100份起买、前收盘 ±10% 涨跌幅边界。
- 连续竞价：市价单、限价单、撤单、价格优先、同价时间优先、冻结资产和当日有效。
- 开盘集合竞价：09:15–09:25 独立订单池、09:20 后撤单锁定、09:30 单一开盘价、未成交转连续竞价。
- 自动交易者统一格式委托账本：记录方向、限价、申请量、成交量、剩余量与状态。
- 可见十档盘口：叠加做市深度、自动订单残余和玩家挂单，并列出来源拆分。
- 玩家账户、冻结现金/持仓、待处理订单、委托日志。
- 浏览器本地存档、导入、导出；智能分析只读资料接口。
- 确定性随机种子、一致性检查与自检脚本。

## 价格主链

```text
国际现货黄金 × 美元兑人民币
→ 人民币换算金价
→ 国内黄金溢价
→ 国内黄金现货价格
→ 基金理论净值
→ 自动交易者与玩家订单
→ 集合竞价 / 连续竞价撮合
→ 二级市场交易价格
```

技术指标不会直接改价格；它们只能影响交易者的后续订单。

## 运行与自检

```text
npm run 自检
```

自检覆盖种子复现、集合竞价锁单与开盘价、连续竞价同价时间优先、冻结资产、撤单、涨跌停边界、现金份额守恒，以及统一委托流和可见十档叠加。

## 当前边界

自动交易者的订单已经进入统一格式账本并出现在可见深度中，但其价格结算仍由现有聚合撮合内核完成。下一步会把自动订单实际迁入中心限价订单簿，替代这层兼容桥接。

## 存档提醒

市场存档保存在当前浏览器的 IndexedDB 和本地存储中。同设备、同浏览器通常可以恢复；换设备、清除站点数据或使用无痕模式时可能丢失。重要世界线请及时导出存档。

当前中国黄金市场状态版本为 `2`；旧版华辰工业存档不能直接导入新市场。
