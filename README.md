# 华辰工业模拟市场

这是一个部署在静态网页托管上的单标的股市模拟市场。

当前只存在一只虚构股票：**华辰工业**。  
它不是随机涨跌演示器。每一次价格变化都由可记录、可复盘的市场状态推进。

## 当前已完成的第一层

- 单一股票的市场状态机
- 内在价值缓慢变化
- 平稳、上行、下行、紧张四种市场状态
- 条件波动率与波动聚集
- 价值资金、趋势资金、噪声资金、风控资金、做市资金
- 流动性、买卖价差、买卖盘与订单冲击
- 随机事件与跳跃波动
- 每一刻的详细波动日志
- 市价买入和卖出；交易也会形成价格冲击并写入日志
- 浏览器本地存档、导入存档、导出存档
- 智能分析扩展边界：只读市场摘要与日志，不能直接改写市场

## 价格如何生成

每一刻推进五分钟虚拟时间。收益率由以下部分相加：

\[
\text{收益率} =
\text{状态漂移} +
\text{价值回归} +
\text{趋势跟随} +
\text{订单冲击} +
\text{随机波动} +
\text{事件冲击}
\]

价格随后按连续复利方式更新：

\[
P_{t+1}=P_t \times e^{r_t}
\]

其中：

- **价值回归**：价格离内在价值越远，回归力量越强；
- **趋势跟随**：短期收益均值高于长期收益均值时，趋势资金偏向买入；
- **订单冲击**：同样的净买卖量，在低流动性下会造成更大冲击；
- **随机波动**：由条件方差决定，因此大波动之后往往仍较动荡；
- **事件冲击**：低概率出现，带来非连续的大幅变化；
- **做市资金**：根据库存压力调整报价与交易方向，维持市场可交易性。

这不会预测真实市场。它追求的是让模拟市场具有真实金融时间序列的部分结构：波动聚集、流动性收缩、趋势延续、均值回归、跳跃风险和订单冲击。

## 存档原则

静态网页没有数据库。市场数据保存在你当前浏览器的本地数据库中。

同一设备、同一浏览器正常刷新或重新打开网页时，市场会从上次状态继续。  
更换浏览器、清除网站数据、使用无痕模式或换设备时，本地存档可能消失。

因此重要世界线应定期点击“导出存档”。导出的文件可通过“导入存档”恢复。

## 智能分析扩展

`智能分析.js` 是智能分析接入边界。

市场内核把以下内容打包为可读资料：

- 当前价格、内在价值、市场状态
- 条件波动率、流动性与买卖价差
- 账户状态
- 最近一段完整波动日志
- 明确的分析约束

智能分析员只能解释资料，不能取得市场状态的写入权限。后续接入外部服务时，密钥不能写进静态网页或仓库。

## 文件结构

- `index.html`：全中文页面结构
- `样式.css`：页面样式与移动端布局
- `核心.js`：市场状态、价格模型、交易和日志
- `存档.js`：浏览器本地存档、导入和导出
- `智能分析.js`：未来智能分析的只读接入边界
- `界面.js`：页面渲染与交互
