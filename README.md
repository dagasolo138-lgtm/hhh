# 中国黄金模拟市场

> **每次开发完成后，必须在 `codex.md` 末尾追加版本更新概述。**

这是一个部署在 GitHub Pages 的纯前端、单标的中国黄金模拟市场。

唯一可交易资产是人民币计价的“中国黄金现货指数基金”模拟份额。交易价格来自国际现货黄金、美元兑人民币、国内黄金溢价、基金理论净值、交易者订单、集合竞价和连续竞价机制。

完整架构、变量定义、因果图、文件职责、存档规则、测试规则、已知限制和版本历史，以 [`codex.md`](./codex.md) 为准。交易制度以 [`交易制度参数表.md`](./交易制度参数表.md) 和 `市场/交易制度参数.js` 为准。

## 当前能力

- 国际金价 → 汇率 → 人民币换算金价 → 国内现货 → 基金理论净值链路。
- 理论净值与二级市场交易价格分离。
- 价值套利、趋势资金、散户群体、风险控制资金、先行信号资金、做市商与玩家进入连续竞价中心限价订单簿。
- 中心订单簿按价格优先、同价时间优先逐笔成交，并执行自成交保护。
- 开盘集合竞价：玩家、五类自动资金和做市背景深度共同形成 09:30 单一开盘价。
- 自动集合竞价订单按开盘价清算；未成交自动意图会登记并接续到后续连续竞价中心撮合。
- 成交直接更新自动交易者和玩家的现金、持仓、平均成本、已实现盈亏、成交价、成交量、订单失衡和中心盘口。
- 可见十档盘口直接读取中心做市深度、自动订单残余和玩家挂单，并列出来源拆分。
- 当日回转、0.001元报价单位、100份起买、前收盘 ±10% 涨跌幅边界。
- 连续竞价：市价单、限价单、撤单、冻结资产、当日有效与跨日失效。
- 开盘集合竞价：09:15–09:25 独立订单池、09:20 后撤单锁定、09:30 单一开盘价、未成交转连续竞价。
- 玩家账户、冻结现金/持仓、待处理订单、委托日志。
- 浏览器本地存档、导入、导出；智能分析只读资料接口。
- 确定性随机种子、一致性检查与自检脚本。

## 价格主链

```text
国际现货黄金 × 美元兑人民币
→ 人民币换算金价
→ 国内黄金溢价
→ 基金理论净值
→ 玩家订单 + 自动交易者开盘/连续订单
→ 开盘集合竞价 / 中心限价订单簿逐笔成交
→ 二级市场交易价格
```

技术指标不会直接改价格；它们只能影响交易者的后续订单。

## 运行与自检

```text
npm run 自检
```

自检覆盖种子复现、自动开盘集合竞价、集合竞价锁单与开盘价、自动与玩家中心撮合、冻结资产、撤单、跨刻保留、涨跌停边界、现金份额守恒、十档来源叠加和自成交保护。

## 当前边界

开盘集合竞价已纳入自动资金，但当前只有一个玩家账户；玩家挂单不会与同一账户自身订单成交。多玩家交叉、同价比例分配和收盘集合竞价尚未实现。

自动订单从开盘集合竞价未成交时会登记为连续竞价接续意图，并在下一刻进入中心撮合；当前接续后会按连续竞价规则重新报价，不保留原开盘委托价格。

## 存档提醒

市场存档保存在当前浏览器的 IndexedDB 和本地存储中。同设备、同浏览器通常可以恢复；换设备、清除站点数据或使用无痕模式时可能丢失。重要世界线请及时导出存档。

当前中国黄金市场状态版本为 `2`；旧版华辰工业存档不能直接导入新市场。
