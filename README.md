# 中国黄金模拟市场

> **每次开发完成后，必须在 `codex.md` 末尾追加版本更新概述。**

这是一个部署在 GitHub Pages 的纯前端、单标的中国黄金模拟市场。

唯一可交易资产是人民币计价的“中国黄金现货指数基金”模拟份额。它通过国际现货黄金、美元兑人民币、国内黄金溢价、基金理论净值、交易者订单和订单簿撮合形成交易价格。

完整架构、变量定义、因果图、文件职责、存档规则、测试规则、已知限制和版本历史，统一以 [`codex.md`](./codex.md) 为准。开发时按需要查阅；每次开发结束后必须更新其中的版本记录。

## 当前能力

- 中国黄金的国际金价 → 汇率 → 人民币换算金价 → 国内现货 → 基金理论净值链路。
- 理论净值与二级市场交易价格分离。
- 价值套利、趋势资金、散户群体、风险控制资金、信息优势资金与做市商。
- 十档买卖盘口、价差、深度、撤单率、补单率、订单失衡和价格冲击。
- 市场事件、隐藏情绪状态、跨期反馈和波动日志。
- 玩家市价买卖；订单会吃掉盘口并改变市场价格。
- 浏览器本地存档、导入、导出。
- 智能分析只读资料接口。
- 确定性随机种子、一致性检查与自检脚本。

## 价格主链

```text
国际现货黄金 × 美元兑人民币
→ 人民币换算金价
→ 国内黄金溢价
→ 国内黄金现货价格
→ 基金理论净值
→ 交易者订单与订单簿撮合
→ 二级市场交易价格
```

技术指标不会直接改价格；它们只影响交易者的后续订单。

## 运行与自检

```text
npm run 自检
```

自检用于检查同种子复现、不同种子差异、价格与净值有效性、订单簿有效性、玩家买卖，以及现金和份额守恒。

## 存档提醒

市场存档保存在当前浏览器的 IndexedDB 和本地存储中。同设备、同浏览器通常可以恢复；换设备、清除站点数据或使用无痕模式时可能丢失。重要世界线请及时导出存档。

当前中国黄金市场状态版本为 `2`；旧版华辰工业存档不能直接导入新市场。
